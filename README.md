# 버튼 잡기 게임 (C# 코딩)

# 개요
-C# 프로그래밍 학습
-핵심기능 : 버튼을 클릭하여 점수를 획득, 게임 오버시 실패
-화면구성 : 창, 다시시작 버튼, 점수 버튼
